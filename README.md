# Left-4-Dead-2-Shop-Mod

This mod is a fork that fixes the issue of multiplayer not working properly for the ShopX mod

original mod:
https://steamcommunity.com/sharedfiles/filedetails/?id=3572463501&searchtext=shopx

Fixes:
- everyone's points no longer gets reset when someone leaves or joins
- everyone's points no longer gets reset when the map changes level

New features:
- you now keep your points across rounds of survival.

Bugs:
- some player get multiple points rewarded for kills, maybe be due to quirks with L4D2 multiplayer connection
